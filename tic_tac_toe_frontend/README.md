# Lightweight React Template for KAVIA

This project now powers a modern Tic Tac Toe game built with React and a professional blue/gold style.

## Features

- Modern, fully accessible 3x3 Tic Tac Toe game in the browser.
- **Ocean Professional theme**: Subtle blue gradients, gold highlights, and clean white surfaces.
- Two-player local play (X vs O), with status updates.
- Visually highlights the winning line.
- Draw/win detection with status and disables moves on finish.
- Full keyboard support, focus outlines, ARIA labels, and screen reader friendly.
- Game state resets and re-centers play at any time.

## Quickstart

1. Install dependencies (if not already):
   ```
   npm install
   ```

2. Run the app:
   ```
   npm start
   ```

   Visit [http://localhost:3000](http://localhost:3000)

3. Run the test suite:
   ```
   npm test
   ```

No API setup is needed; no network calls occur.
